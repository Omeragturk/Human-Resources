        <h2>Human Resources</h2>
        <p>This project is a web application developed using ASP .NET Core MVC and Entity Framework Core with the Code
            First approach. The project has been developed within the framework of Scrum, one of the most important
            methodologies of the Agile approach. It consists of two separate projects, structured as API and MVC.</p>

        <h2>Technologies Used</h2>

        <ul>
            <li>ASP .NET Core MVC: Used for the user interface and server-side operations of the application.</li>
            <li>Entity Framework Core: Employed with the Code First approach for database operations.</li>
            <li>Scrum: Utilized for project management and development processes.</li>
        </ul>

        <h2>Project Roles and Features</h2>

        <h3>Site Manager</h3>
        <ul>
            <li>Homepage: Main control panel for the site manager.</li>
            <li>Membership and Login Page: Allows users to log in to the system.</li>
            <li>Member Settings Page: Page for managing user information and settings.</li>
            <li>Companies Page: Used for managing company information.</li>
            <li>Definitions Page: Page for various system-wide definitions.</li>
        </ul>

        <h3>Company Manager</h3>
        <ul>
            <li>Homepage: Main control panel for the company manager.</li>
            <li>Membership and Login Page: Allows users to log in to the system.</li>
            <li>Member Settings Page: Page for managing user information and settings.</li>
            <li>Personnel Management Page: Used for managing personnel information.</li>
            <li>Personnel CRUD Operations: Operations for adding, updating, and deleting personnel.</li>
            <li>Enable/Disable Employee: Operations to activate or deactivate personnel.</li>
            <li>Personnel Document CRUD Operations: Management of personnel documents.</li>
            <li>Manage Permissions Page: Management of personnel permissions.</li>
            <li>Expense Creation Page: Management of expense operations.</li>
            <li>Add/Delete/Update Inventory Page: Management of personnel inventory.</li>
        </ul>

        <h3>Personnel</h3>
        <ul>
            <li>Homepage: Main control panel for personnel.</li>
            <li>Membership and Login Page: Allows users to log in to the system.</li>
            <li>Member Settings Page: Page for managing user information and settings.</li>
            <li>Leave Page: Management of personnel leave requests.</li>
            <li>Inventory Management Page: Management of personnel inventory.</li>
        </ul>
